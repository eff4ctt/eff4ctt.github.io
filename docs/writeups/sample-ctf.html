The Challenge

We were given a binary and the source code.

#include 
#include 

void win() {
    printf("You won!\n");
}

void vuln() {
    char buffer[64];
    gets(buffer);
}

int main() {
    vuln();
    return 0;
}

The Solution

The `gets` function is vulnerable to buffer overflow. We can overwrite the return address to jump to `win`.

Steps: 1. Find the offset. 2. Find the address of `win`. 3. Construct the payload.
